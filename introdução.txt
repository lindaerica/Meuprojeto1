Clone:

Clonar um repositório Git é o processo de criar uma cópia local de um repositório remoto. Isso é feito usando o comando git clone seguido da URL do repositório remoto.
O comando cria uma pasta local contendo todos os arquivos e histórico de commits do repositório remoto.
Merge:



O merge é usado para combinar as alterações de duas ou mais branches em um único branch.
O comando git merge permite integrar as alterações de uma branch em outra. Isso pode ser feito de forma automática ou manual, dependendo das circunstâncias.
O objetivo é garantir que todas as alterações sejam incorporadas de forma coesa e sem conflitos.
Push:

O push é usado para enviar as alterações locais para um repositório remoto.
O comando git push é usado para atualizar o repositório remoto com as alterações que foram feitas na sua cópia local.
Isso permite que outros colaboradores acessem suas alterações e mantém o repositório remoto atualizado
